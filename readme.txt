Please execuate "src/Crawler.java", then execuate "htdocs/start-server.cmd".
Please remember to terminate your previous opened CGI before execuation.

"sponsor.txt" contains all the web address which are sponsors, it will affect the ranking algorithm if the URL belongs to sponsor group.
All files of keywords are stored at "C:\db"


**Make sure you don't use "C:\db" as any application or file storage before execuation of this program (everything inside will be erased)

Developed in Java8 and Python 3.5.2

Suggested to run with Eclipse IDE due to the .class file generation related to the CGI server.
(If compilied through CMD or Terminal, search.py needs to be modified before running the CGI in order to get the result.)
